# ENVS 193DS Infographic

Spring 2026

## General Information

This repo provides examples of creating different figures/plots to display 
personal data in a variety of ways. 

The personal data in this repo examines the relationship between my daily water 
intake (liters) and a variety of different variables, including (but not 
limited to ) hours I worked in a shift that day, daily steps, and sickness.

The figures created are a jitter plot, a bubble plot, a ridge line plot, and
a pie chart. 

All of these figures were then used to create an infographic, the link for
which is provided below.

At the end, there are also written components on the process of creating these
visualizations: inspiration, aesthetic choices, data cleaning explanation, and
pieces of my thoughts during the process. 

## Data and file information

There is a 'data' folder in which my personal data is stored.

There is a 'code' folder in which all of my code for the visualizations is
stored.

There is a 'visualizations' folder in which all of my figures are stored.
